# University Management System (Java)

## Overview
This project is a console-based **University Management System** developed in Java.  
It manages students and courses efficiently using different **Data Structures** concepts such as:

- Binary Search Tree (BST)
- Queue
- Linked List
- Stack

The project demonstrates how data structures can be used in real-world management systems.

---

## Features

### Student Management
- Add new students
- Search students
- Store and organize students using Binary Search Tree
- Manage student records efficiently

### Course Management
- Add and manage courses
- Store course data using linked structures
- Course enrollment handling

### Waitlist Management
- Queue implementation for course waitlists
- Students can be placed in waiting queue when seats are full

### Undo Functionality
- Stack-based undo operations
- Reverse recent actions

---

## Data Structures Used

| Data Structure | Purpose |
|---------------|---------|
| Binary Search Tree | Student record management |
| Queue | Waitlist handling |
| Stack | Undo operations |
| Linked List | Course storage and management |

## Project Files

- `UniManage.java` → Main driver program  
- `Student.java` → Student class  
- `StudentNode.java` → Student node implementation  
- `StudentBST.java` → Binary Search Tree for students  
- `Course.java` → Course class  
- `CourseNode.java` → Course node implementation  
- `CourseList.java` → Course linked list management  
- `WaitlistQueue.java` → Queue implementation for waitlist  
- `UndoStack.java` → Stack implementation for undo  
- `QNode.java` → Queue node structure
